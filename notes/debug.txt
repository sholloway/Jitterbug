Tools to use for profiling on the command line
	time
		outputs:
			real -> clock time
			user -> user time
			sys -> kernel time
	
	dtruss
		sits on top of dTrace.
		To use:
			sudo dtruss /Users/sholloway/Dropbox/Jitterbug/Foxtrot.app/Contents/MacOS/Foxtrot render
